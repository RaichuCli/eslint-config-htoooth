# htoo eslint config

## Usage
使用之前必须安装：

```bash
npm install --save-dev eslint-config-htoooth
```

使用方法：
1. 在项目根目录下新建文件： `.eslintrc`
2. 写入以下文件：
```json
{
  "extends": "htoooth"
}
```
